# 身内グルメマップ (Internal Gourmet Map)

身内（仲の良いメンバー）だけで、おすすめの飲食店情報を共有・蓄積するためのプライベートなWebアプリケーションです。

## 主な機能
- **簡易認証ログイン**: 合言葉を知っている人のみ閲覧・投稿が可能
- **店舗投稿機能**: 店名、カテゴリ、評価、予算、隠れ家度、推しポイントを投稿可能
- **一覧・検索機能**: カテゴリや予算での絞り込み検索
- **Google Maps 連携**: 詳細画面から1タップでGoogle Maps検索
- **「行ったよ！」機能**: 気になるお店にリアクション可能

## 技術スタック
- **Backend**: Java 17+, Spring Boot 3.x
- **Database**: H2 Database (ファイルベース)
- **Frontend**: Thymeleaf, Vanilla CSS
- **Testing**: Cypress (E2E)

## 開発環境のセットアップ

### バックエンドの起動
```bash
# プロジェクトルートで実行
mvn spring-boot:run
```
起動後、ブラウザで `http://localhost:8080` にアクセスしてください。
- **合言葉**: `qa202601` (デフォルト設定)

### テストの実行 (Cypress)
```bash
npm install
npm run test:desktop
```

## 運用とバックアップについて

本アプリケーションはシンプルに動かすため、**H2 Databaseをファイルベース**で利用しています。
投稿データはアプリケーションルートディレクトリ直下の `data/gourmetdb.mv.db` に保存されます。

**【バックアップ方法】**
定期的に `data/gourmetdb.mv.db` ファイルを安全な場所（クラウドストレージなど）にコピーしてください。これだけでデータのバックアップが完了します。

## 構成のカスタマイズ
- `src/main/resources/application.properties` にて、アップロード上限やデータベース設定を変更可能です。
- 本番環境（Render等）で動かす場合も、パスワードなどを環境変数で切り替えやすく作られています。
